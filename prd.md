# 产品需求文档 (PRD)：AI 编程新手通关手册网站

## 1. 项目概述

* **项目目标** ：将《新手入门指南》PDF 转化为一个结构化的 Web 学习平台。
* **核心价值** ：解决新手“不知道从哪开始”、“环境配置难”、“报错不敢问”的痛点。
* **设计风格** ：极简技术文档风（参考 Vite/Next.js 官网），支持深色/浅色模式切换。

## 2. 核心架构与页面逻辑

### 2.1 侧边栏导航 (Sidebar)

网站按 PDF 逻辑分为三个阶段，由浅入深：

* **第一阶段：环境准备 (Setup)**
  * 科学上网指引
  * IDE 选择：Trae vs Cursor (对比与下载)
* **第二阶段：入门实战 (Web Dev)**
  * 开发第一个网页 (3选1 任务)
  * 环境五件套 (Node.js, Git, Python 等)
  * 一键上线：Vercel 部署指南
* **第三阶段：高级进阶 (Claude Code)**
  * Claude Code 安装与配置
  * CC Switch 与 API 接入 (Univibe)
  * 自动化实战案例
* **附录：新手避坑指南**
  * 报错提问模板
  * 资源下载汇总

## 3. 功能模块详细说明

### 3.1 环境选择模块 (Step 1)

* **内容要求** ：清晰对比 Trae (国内免魔法) 与 Trae 国际版/Cursor 的优劣。
* **组件** ：对比表格，提供直接跳转官网的下载按钮。

### 3.2 任务导向模块 (Step 2)

* **实战选题卡片** ：

1. 图片字幕生成器
2. 摸鱼小游戏
3. 朋友圈五宫格分析

* **部署 Checklist** ：清晰列出安装 Node.js、Git、Github 的步骤，并配有对应的“验证命令”（如 `node -v`）。

### 3.3 Claude Code 专题模块 (Step 3)

* **逻辑可视化** ：使用清晰的文字或图示解释：
* **API Key** = 话费/流量
* **Claude Code** = 手机硬件
* **CC Switch** = 运营商切换器/开关
* **配置手册** ：一步步引导用户从 Univibe 获取 Key，到终端输入命令完成配置。

### 3.4 交互式代码组件

* **一键复制** ：所有终端命令（如 `npm install -g @anthropic-ai/claude-code`）必须有一键复制按钮。
* **高亮警告框** ：用于提示“请勿对外传播民间教学链接”或“科学上网合法用途”等重要警告。

---

## 4. 技术需求 (Technical Requirements)

* **响应式设计** ：完美适配 PC 端（查阅文档）和移动端（快速预览）。
* **搜索功能** ：集成简单的全文搜索，方便查找特定工具的配置。
* **性能要求** ：纯静态页面加载，确保弱网环境下（尤其配置环境时）也能快速打开。

---

## 5. 核心文案摘要（摘自 PDF）

* **核心逻辑** ：先让 AI 写一份说明书 (PRD)，再让它动手写代码。
* **心态建设** ：终端只是一个对话框，不要怕黑框框；遇到红色报错直接截图问 AI。
* **作业打卡** ：完成作业后发送链接至群聊并 @小哲教练 检查。
